# spotify-clone

Project Title: # Spotify Web Player Clone — A responsive music streaming interface built with HTML and CSS.

Project Goal: This project focuses on replicating the core UI/UX of the Spotify desktop application, including the sidebar and player controls.

Layout System: The interface utilizes a Flexbox layout to manage the sidebar, main content area, and a fixed-position music player.

Navigation Features: Includes a sticky top navigation bar and a dynamic sidebar with library and playlist management sections.

Iconography & Fonts: Integrated Font Awesome for UI icons and the Montserrat Google Font for a modern, authentic aesthetic.

Interactive Elements: Features styled custom range inputs for the playback bar and hover effects on navigation options.

Responsive Design: Implemented CSS Media Queries to ensure the layout adapts to different screen sizes and hide specific elements on smaller viewports.
